# KaamChore

## Description

A Cross-Platform application to manage house chore among groups and members.

## Authors

- [Darshan Nariya](https://www.github.com/dannyglade)

## Features

- Create Groups
- Invite Members
- Create Chores
- Assign Chores
- Add Chore Deadline
